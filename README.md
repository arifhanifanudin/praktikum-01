" MENENTUKAN LUAS PERSEGI PANJANG "

 Deskripsi
 
 1. Input panjang
 
 2. Input lebar
 
 3. Luas = panjang x lebar
 
 4. Print luas
 
 # Program C++
 
 #include <iostream>

using namespace std;

int main()

{

    int p, l, L;
